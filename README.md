# Spring Boot MVC
> 06/17 Training in GLSoft for `Java Spring` framework.
## Description

- Setup Spring Boot MVC building system
    - a. Set up the `pom.xml` configuration.
    - b. Configure the `Thymeleaf` template engine.
-  Design an MVC controller for dispatching operations on the home page.
- Design an order query form using the MVC architecture
    - where the page needs to send the order id form field to the corresponding Controller-Action.

## Demo

run the `mvn spring-boot:run` command in the project root directory, and then open the browser to access the `http://localhost:8080/` address.
